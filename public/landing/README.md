# Landing page images

Add the image files listed in **docs/landing-images.md** to this folder.

Example: `hero.jpg`, `dashboard.jpg`, `clarity.jpg`, `spare-score.jpg`, `budgets.jpg`, `planning.jpg`, `reports.jpg`, `family.jpg`, `trust.jpg`.

Use JPG or WebP. If a file is missing, the landing page shows a placeholder with the filename to add.
