# User Financial Data Inputs

This document lists, by category, the financial data users can enter.

## Accounts
- Name
- Type
- Credit limit
- Initial balance
- Owner IDs
- Due day of month
- Currency code

- Date
- Type (expense, income, transfer)
- Amount
- Account
- To account (transfer destination)

- Subcategory
- Merchant
- Description
- Recurring flag
- Recurring frequency
- Expense type (fixed, variable)
- Receipt URL

- Category name

- Subcategory logo URL
